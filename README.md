<h1>DSMovie</h1>
<h2>Apresentação</h2>
<p>Projeto desenvolvido durante o evento Semana Spring React da plataforma Devsuperior.<br>
A aplicação consiste em um catálogo de filmes com paginação e registro de avaliações dos usuários</p>
